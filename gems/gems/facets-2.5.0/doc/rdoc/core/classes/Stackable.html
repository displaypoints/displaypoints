Stackable

Stackable mixin provides pop, push, pull, etc. It depends on slice, splice and insert.

Methods

peek()

Peek at the top of the stack. 

  a = [1, 2, 3]
  a.peek          #=> 3
  a               #=> [1, 2, 3]
